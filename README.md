# BidyaPith

https://zippy-cendol-b9b2c6.netlify.app/

## Project Purpose

**BidyaPith** is an online learning platform designed to help users acquire new skills across multiple domains such as coding, communication, yoga, and more. The platform provides interactive learning resources, skill-based courses, and real-world practical knowledge to empower learners.

## Key Features

- **User Authentication**: Users can register, log in, and log out using email/password or Google login.
- **Profile Management**: Users can view and update their profile information, including name and profile picture.
- **Hero Slider**: Interactive hero slider showcasing platform highlights and motivational content.
- **Testimonials Section**: Displays real user feedback with smooth scroll animations.
- **Responsive Design**: Fully responsive and optimized for desktop, tablet, and mobile devices.
- **Interactive UI Components**: Buttons, forms, and cards with animations and visual feedback.

## NPM Packages Used

- `react` & `react-dom` – Core React library for building UI components.
- `react-router-dom` – For client-side routing and navigation.
- `firebase` – Backend services for authentication and user management.
- `react-icons` – Prebuilt icons for UI enhancements.
- `swiper` – Responsive slider/carousel for hero section.
- `aos` – Animate On Scroll library for smooth scroll animations.
- `tailwindcss` & `daisyui` – Utility-first CSS framework for styling components.
